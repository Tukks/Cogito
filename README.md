# Cogito

## Features

All you notes in one places

- add Link
- add Tweet
- add Note
- add Tag
- Search in Article, Tweet and Note
- PWA app

## Launch dev

### Back

- add application-secret.properties with : 

```
api.twitter.key=
api.twitter.key.secret=
```

- Launch CogitoBackApplication.java

### Front

- npm i
- npm start
